    JAVA DEVELOPMENT ENVIRONMENTS

    Eclipse is one of the oldest Java IDEs around, and is the first IDE that I ever used! It is robust and has many features, however, there are more modern variations of this software.

    IntelliJ IDEA is the Java IDE for those who are serious developers. It is feature rich and is arguably one of the best development environments around.

    PYTHON DEVELOPMENT ENVIRONMENTS

    Pycharm is the best IDE if you are looking to develop in Python.

    C AND C++ DEVELOPMENT ENVIRONMENTS

    CLion is my go to IDE choice when it comes to programming in C and C++.

    CodeBlocks is a great IDE for both C and C++, and is comparable to eclipse in terms of form and function.

    Eclipse. Yes. I know what you are thinking, "Didn't you list eclipse as a Java editor? Indeed I did, and I am listing it here as well because it is also able to develop in C and C++, given you install the proper version of it.

    ALL HAIL TO THE KING

    Notepad++, While lacking many features a full fledged IDE has, is still a great lightweight choice for developing in any programming language! Just be sure you know what you're doing!
